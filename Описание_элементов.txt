Описание сайта:
Сайт предназначен для людей, которые начинают изучать программирование по подписке (месячная, годовая, для образовательных учреждений). Целевая аудитория: школьники и студенты, а так же молодежь до 30 лет. Сайт двуязычный: казахский и русский с возможностью выбора языка. На главной странице находятся все разделы навигации (Курсы, О нас, Цены, Вакансии, Проекты наших учеников). Если пользователь залогинен, то раздел курсов открывается вместо главной страницы. На начальном этапе он состоит из 3 вводных курсов (HTML, CSS и JS). Курсы разделены по модулям, в каждом модуле есть задания. По окончании курса ученик получает сертификат. Пользователь может видеть свой прогресс на странице отдельного курса. Сайт разрабатывается только на фронте, без бэкенда.

Описание элементов на сайте:
1. Главная страница. Страница устроенная как сайт-визитка длинная страница с кнопками навигации по этой же странице. Ее задача сформировать у нового посетителя представление о нас и мотивировать его приобрести подписку на наши курсы.
1.1. Шапка:
    1.1.1. Большая картинка которая является фоном для шапки страницы
    1.1.2. В верхней левой части шапки строка с навигацией на разделы страницы:
        1.1.2.1. Логотип
        1.1.2.2. Курсы
        1.1.2.3. О нас
        1.1.2.4. Цены
        1.1.2.5. Вакансии
        1.1.2.6. Проекты наших учеников
    1.1.3. В верхней правой части шапки строка с кнопками:
        1.1.3.1 Смена языка
        1.1.3.2. Авторизация
        1.1.3.3. Регистрация
    1.1.4. В нижней левой части шапки текст, который появляется с анимацией, словно его печатают
1.2. Основная часть сайта
    1.2.1 Курсы. Тут представленны курсы, которые мы предоставляем в строку. На стандартный экран пользователя должно помещаться без прокрутки 6 курсов. Все в минималистичном виде, при навидении мышы на элементы должно появляться объяснение
        1.2.1.1 Название курса (пример. "Web для начинающих")
        1.2.1.2 Используемый стэк (пример. "HTML", "CSS", "JavaScript")
        1.2.1.3 Количество модулей в курсе (пример. "53")
    1.2.2. О нас. Небольшой текст, описывающий наш проект, для кого он сделан и дающий представление как его использовать
    1.2.3. Цены. Раздел с видами подписок и ценами на них. 3 варианта на выбор пользователя.
        1.2.3.1 Индивидуальная подписка на месяц
            1.2.3.1.1. Название "Индивидуальная подписка на месяц"
            1.2.3.1.2. Продолжительность "1 месяц"
            1.2.3.1.3. Услуги "Доступ ко всем курсам на платформе"
            1.2.3.1.4. Цена "4999 тенге"
            1.2.3.1.5. Кнопка "Купить подписку"
        1.2.3.2 Индивидуальная подписка на год
            1.2.3.2.1. Название "Индивидуальная подписка на месяц"
            1.2.3.2.2. Продолжительность "1 месяц"
            1.2.3.2.3. Услуги "Доступ ко всем курсам на платформе"
            1.2.3.2.4. Цена "49999 тенге"
            1.2.3.2.5. Кнопка "Купить подписку"
        1.2.3.3 Для образовательных учреждений
            1.2.3.3.1. Название "Для образовательных учреждений"
            1.2.3.3.2. Продолжительность "1 год"
            1.2.3.3.3. Услуги "Доступ ко всем курсам на платформе"
            1.2.3.3.4. Цена "4999 тенге"
            1.2.3.3.5. Кнопка "Связаться с нами"
    1.2.4. Вакансии. Раздел с вакансиями, которые требуются нашему проекту. Изначально будет пустой с предупрждением "Актуальных вакансий в данный момент нет. Пожалуйста, проверьте позже"
    1.2.5. Проекты наших учеников. Также изначально пустой. Должно быть предупреждение "Здесь будут публиковаться реальные рабочие проекты наших учеников. Мы недавно запустились, но уже ждем их с нетерпением"
2. Страница "Мои курсы". Это главная страница ученика. Авторизованный пользователь попадает на эту страницу по умолчанию как на главную страницу.
    2.1. Шапка страницы.
        2.1.1. Логотип
        2.1.2. Кнопка смены языка 
        2.1.3. Выйти 
    2.2. Мои курсы. Список карточек курсов, которые начал ученик. Должно быть название, количество пройденных уроков и количество уроков всего в курсе. 
    2.3. Доступные курсы. Список карточек курсами, которые ученик еще не начал. Название, количество уроков в курсе. Карточка должна быть серой, чтобы это отражало ее неактивность
3. Страница курса
    3.1. Название курса (прим. "Web для начинающих")
    3.2. Прогресс бар под названием курса. Должен заполняться в зависимости от количества пройденных модулей
    3.3. Список модулей. Находится в левой части страницы с скролится отдельно от остальных элементов страницы. Каждая страка состоит из (названия модуля "Модуль 1. Hello, World!", количество заданий в модуле)
    3.4. Сертификат об прохождении курса с именем и фамилией ученика. Это изображение-pdf сертификата, который будет выдан ученику после завершения курса. Во время прохождения курса, ученик видит его миниатюру. После прохождения курса при нажатии на нее сертификат можно будет скачать.
4. Страница модуля. Страница на которой происходит обучение. Делится на 3 части, которые скролятся независимо друг от друга.
    4.1. Часть страницы, на которой отображается список заданий и тексты выбранного задания
        4.1.1. Название модуля
        4.1.2. Список заданий в модуле "Задание 1", "Задание 2", "Задание 3". Между которыми пользователь может переключаться.
        1.1.3. Задание. Содержит текст, может содержать видео, изображения.
    4.2. Часть страницы, где учиник пишет/исправляет код по заданию. Есть переключаемые вкладки файлов, которые он может редактировать по данному заданию.
    4.3. Часть страницы, где ученик видит результат своего кода. Вывод консоли или web-страницу. В нижней части кнопка "Подсказка от ИИ", которая и "Далее". Если ответ ученика правильный, открывается следующее задание. Если данное задание последнее в модуле, следующий модуль.
