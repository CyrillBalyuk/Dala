Сабақ 1: HTML негіздері
Түсіндірме:
HTML (HyperText Markup Language) — веб-беттерді құруға арналған тіл. Әр HTML құжаты мынадай құрылымнан басталады:
* <!DOCTYPE html> — құжат түрін көрсетеді.
* <html> — барлық HTML-дің қаптамасы.
* <head> — бет туралы ақпарат (атауы).
* <body> — көрінетін мазмұн.
Тапсырмалар:
1. <html>, <head> және <body> тегтері бар HTML құжатын жасаңыз.
2. <head> тегінде тақырыпты «Менің алғашқы бетім» етіп орнатыңыз.
3. <body> тегіне «Hello, World!» деп жазыңыз.
Дұрыс жауап:


<!DOCTYPE html>
<html>
<head>
  <title>Менің алғашқы бетім</title>
</head>
<body>
  Hello, World!
</body>
</html>


Сабақ 2: Тақырыптар
HTML тілінде алты тақырып тегі бар: <h1> - ең үлкені және <h6> - ең кішісі. Олар тақырыптар, тақырыпшалар және бөлімдер сияқты мазмұнды құрылымдауға көмектеседі. 
Тапсырмалар:
   1. <h1> тегін жасаңыз: «Менің сайтыма қош келдіңіз».
   2. <h2> тегін қосыңыз: «Мен туралы».
   3. <h3> тегін қосыңыз: «Хобби».
Дұрыс жауап:


<!DOCTYPE html>
<html>
<body>
  <h1>Менің сайтыма қош келдіңіз</h1>
  <h2>Мен туралы</h2>
  <h3>Хобби</h3>
</body>
</html>


Сабақ 3: Абзацтар мен жол үзілімдері
Абзацтар <p> тегі арқылы жазылады. Абзац ішіндегі жолды үзу үшін <br> тегін пайдаланыңыз. Бірнеше <p> тегтері бөлек мәтін блоктарын жасайды.
      1. <p> арқылы өзіңіз туралы абзац жазыңыз.
      2. Абзац ішінде <br> арқылы сөйлемді екі жолға бөліңіз.
      3. Кез келген мәтінмен екінші абзацты қосыңыз.
<!DOCTYPE html>
<html>
<body>
  <p>Сәлем! Менің атым Арсен.<br>Маған код жазу ұнайды!</p>
  <p>Бұл менің екінші абзацым.</p>
</body>
</html>


Сабақ 4: HTML-де мәтінді пішімдеуге арналған тегтер бар:
<b> → қалың
<i> → курсив
<u> → астын сызу
Тапсырмалар:
<b> көмегімен қалың мәтінді жазыңыз.
<i> арқылы курсив мәтін жазыңыз.
<u> арқылы асты сызылған мәтінді жаз.
Дұрыс жауап:


<!DOCTYPE html>
<html>
<body>
  <p><b>Қалың мәтін</b></p>
  <p><i> Kурсив мәтін </i></p>
  <p><u>Aсты сызылған мәтін</u></p>
</body>
</html>


Сабақ 5: Тізімдер ақпаратты ұйымдастыруға көмектеседі:
         1. <ul> = таңбаланған тізім (нүктелер).
         2. <ol> = нөмірленген тізім (сандар).
         3. <li> = тізім элементі. Тізімдерді кірістіруге болады.
Тапсырмалар:
1. Үш жемістің маркерленген тізімін жасаңыз.
2. Үш элементтің нөмірленген тізімін жасаңыз.
3. Бірінші тізімді екіншісіне кірістріңіз.
Дұрыс жауап:


<!DOCTYPE html>
<html>
<body>
  <ul>
    <li>Алма</li>
    <li>Банан</li>
    <li>Апельсин</li>
  </ul>


  <ol>
    <li>Бірінші</li>
    <li>Екінші</li>
    <li>Үшінші</li>
  </ol>


  <ul>
    <li>Сусындар
      <ol>
        <li>Су</li>
        <li>Шырын</li>
      </ol>
    </li>
  </ul>
</body>
</html>


Сабақ 6: Сілтемелер <a> арқылы жасалады. href атрибуты мекенжайды көрсетеді. Сілтемені жаңа қойындыда ашу үшін target="_blank" пайдаланыңыз. Сондай-ақ жергілікті файлдарға сілтеме жасай аласыз (мысалы, about.html).
Тапсырмалар:
1. ”Google" мәтінімен https://google.com сілтемесін жасаңыз.
2. Жаңа қойындыда ашылатын сілтеме жасаңыз.
3. Басқа about.html бетіне сілтеме қосыңыз.
<!DOCTYPE html>
<html>
<body>
  <a href="https://google.com">Google</a><br>
  <a href="https://google.com" target="_blank">Google в новой вкладке</a><br>
  <a href="about.html">О сайте</a>
</body>
</html>


Сабақ 7: Суреттер <img> арқылы қосылады.
src — файл жолы
alt — сипаттама (қол жетімділік үшін маңызды)
width / height - ені/биіктігі
Суретті <a> ішіне орау арқылы сілтеме жасауға болады.
Тапсырмалар:
cat.jpg суретін alt="Сүйкімді мысық" арқылы қосыңыз.
Енді 200 пиксельге орнатыңыз.
Суретінді басуға болатын етіп жасаңыз және Google-ге сілтеме жасаңыз.
<!DOCTYPE html>
<html>
<body>
  <img src="cat.jpg" alt="Сүйкімді мысық" width="200">
  <a href="https://google.com">
    <img src="cat.jpg" alt="Басуға болатын мысық" width="200">
  </a>
</body>
</html>


Сабақ 8: Кестелер:
Кестелер деректерді жолдар мен бағандарда көрсетеді.
<table> — кесте
<tr> — жол
<th> — тақырып
<td> — деректер ұяшығы
Тапсырмалар:
1. 2 жол және 2 баған бар кесте құру.
2. <th> арқылы тақырыптарды қосыңыз.
3. <td> арқылы деректер жолын қосыңыз.
<!DOCTYPE html>
<html>
<body>
  <table border="1">
    <tr>
      <th>Аты</th>
      <th>Жасы</th>
    </tr>
    <tr>
      <td>Арсен</td>
      <td>25</td>
    </tr>
  </table>
</body>
</html>


Сабақ 9: Формалар
Формалар пайдаланушылардан деректерді жинауға мүмкіндік береді.
   1. <input type="text"> — мәтіндік өріс
   2. <input type="password"> — құпия сөз өрісі
   3. <input type="submit"> — жіберу түймесі
Тапсырмалар:
      1. Аты өрісі бар форма жасаңыз.
      2. Құпия сөз өрісін қосыңыз.
      3. Жіберу түймешігін қосыңыз.
<!DOCTYPE html>
<html>
<body>
  <form>
    Имя: <input type="text" name="username"><br>
    Пароль: <input type="password" name="password"><br>
    <input type="submit" value="Жіберу">
  </form>
</body>
</html>


Сабақ 10: Семантикалық тегтер
Семантикалық тегтер мазмұнның мағынасын сипаттайды:
<header> = тақырып
<nav> = навигация мәзірі
<footer> = төменгі бөлігі (сайт жертөлесі)
Бұл SEO және қол жетімділікті жақсартады.
Тапсырмалар:
1. Сайт тақырыбымен <тақырып> жасаңыз.
2. Навигация сілтемелері бар <nav> жасаңыз.
3. Авторлық құқық мәтіні бар <foooter> қосыңыз.


<!DOCTYPE html>
<html>
<body>
  <header>
    <h1>Менің сайтым</h1>
  </header>


  <nav>
    <a href="index.html">Басты</a> |
    <a href=“about.html">Мен туралы</a>
  </nav>


  <footer>
    <p>&copy; 2025 Менің сайтым</p>
  </footer>
</body>
</html>


Сабақ 11: Қорытынды жоба — «Мен туралы» бет
Сіз HTML тілінің негіздерін үйрендіңіз: құрылым, тақырыптар, мәтін, тізімдер, сілтемелер, суреттер, кестелер, пішіндер және семантика. Енді осы білімнің барлығын пайдалана отырып, толық жұмыс істейтін «Мен туралы» бетін жасаңыз.
Тапсырмалар:
         1. <!DOCTYPE html>, <html>, <head>, <body> көмегімен HTML құрылымын жасаңыз.
         2. Атыңыз <h1> және мәзір <nav> бар тақырыпты <header> қосыңыз.
         3. Маңызды сөздерді қою және курсивпен белгілеп, өзіңіз туралы абзац жазыңыз.
         4. Суретті кірістіру (мысалы, me.jpg).
         5. Хоббилеріңізді тізімдеңіз.
         6. Аты-жөніңіз бен жасыңызды көрсететін кесте жасаңыз.
         7. Байланыс формасын қосыңыз (аты, электрондық поштасы, хабарлама, түйме).
         8. Төменгі жағында <footer> қосыңыз.
Дұрыс жауап:
<!DOCTYPE html>
<html>
<head>
  <title>Мен туралы</title>
</head>
<body>


  <header>
    <h1>Арсен Аманбай</h1>
    <nav>
      <a href="index.html">Басты</a> |
      <a href="#hobbies">Хобби</a> |
      <a href="#contact">Контакілер</a>
    </nav>
  </header>
  <section>
    <h2>Мен туралы</h2>
    <p>Сәлем! Менің атым <b>Арсен Аманбай</b>. Мен бастаушы веб-әзірлеушімін және 
    <i>жаңа технологияларды үйренуді ұнатамын</i>.</p>
  </section>


  <!-- Сурет -->
  <section>
    <h2>Менің суретім</h2>
    <img src="me.jpg" alt=“Мен туарлы сурет" width="200">
  </section>


  <!-- Хобби -->
  <section id="hobbies">
    <h2>Менің хобби</h2>
    <ul>
      <li>Бағдарламау</li>
      <li>Гитарада ойнай</li>
      <li>Саяхаттау</li>
    </ul>
  </section>


  <!-- Кесте -->
  <section>
    <h2>Негізгі ақпарат</h2>
    <table border="1">
      <tr>
        <th>Аты жөні</th>
        <td>Арсен Аманбай</td>
      </tr>
      <tr>
        <th>Жасы</th>
        <td>25</td>
      </tr>
    </table>
  </section>


  <!-- Форма -->
  <section id="contact">
    <h2>Маған хабарласу</h2>
    <form>
      Аты: <input type="text" name="name"><br>
      Email: <input type="email" name="email"><br>
      Хабарлама: <br><textarea name="message" rows="4" cols="30"></textarea><br>
      <input type="submit" value="Жіберу">
    </form>
  </section>


  <!-- Жертөле -->
  <footer>
    <p>&copy; 2025 Мой сайт</p>
  </footer>


</body>
</html>
