Базовая структура HTML


1. Каждый HTML-документ начинается со структуры. Тег <!DOCTYPE html> сообщает браузеру, что это документ HTML5. Тег <html> оборачивает всё. Внутри него находятся теги <head> (метаданные, например, заголовок) и <body> (видимое содержимое страницы).


   1. Создайте HTML-документ с тегами <html>, <head> и <body>.
1. В теге <head> задайте заголовок «Моя первая страница».
2. В теге <body> напишите «Hello, World!».


<!DOCTYPE html>
<html>
<head>
  <title>Моя первая страница</title>
</head>
<body>
  Hello, World!
</body>
</html>


2. Заголовки
В HTML есть 6 тегов заголовков: <h1> — самый большой, <h6> — самый маленький. Они помогают структурировать контент, например, заголовки, подзаголовки и разделы.Задачи:
   1. Создать тег <h1> «Добро пожаловать на мой сайт».
   2. Добавить тег <h2> «Обо мне».
   3. Добавить тег <h3> «Хобби».
<!DOCTYPE html>
<html>
<body>
  <h1>Добро пожаловать на мой сайт</h1>
  <h2>Обо мне</h2>
  <h3>Хобби</h3>
</body>
</html>


3. Параграфы и переносы строк
Абзацы пишутся с помощью тега <p>. Для переноса строки внутри абзаца используйте тег <br>. Несколько тегов <p> создают отдельные текстовые блоки.


      1. Напишите абзац о себе, используя <p>.
      2. Внутри абзаца разделите предложение на две строки, используя <br>.
      3. Добавьте второй абзац с любым текстом.
<!DOCTYPE html>
<html>
<body>
  <p>Привет, меня зовут Арсен.<br>Мне нравится писать код!</p>
  <p>Это мой второй параграф.</p>
</body>
</html>


4. HTML имеет теги для оформления текста:
* <b> → жирный
* <i> → курсив
* <u> → подчёркнутый
Задания:
1. Напишите жирный текст с помощью <b>.
3. Напишите курсив с помощью <i>.
4. Напишите подчёркнутый текст с помощью <u>.
<!DOCTYPE html>
<html>
<body>
  <p><b>Жирный текст</b></p>
  <p><i>Курсивный текст</i></p>
  <p><u>Подчёркнутый текст</u></p>
</body>
</html>


5. Списки помогают структурировать информацию:
* <ul> = маркированный список (точки).
* <ol> = нумерованный список (цифры).
* <li> = элемент списка.
Списки можно вкладывать друг в друга.
Задания:
   1. Создайте маркированный список из трёх фруктов.
   5. Создайте нумерованный список из трёх пунктов.
   6. Вложите один список в другой.
<!DOCTYPE html>
<html>
<body>
  <ul>
    <li>Яблоко</li>
    <li>Банан</li>
    <li>Апельсин</li>
  </ul>


  <ol>
    <li>Первый</li>
    <li>Второй</li>
    <li>Третий</li>
  </ol>


  <ul>
    <li>Напитки
      <ol>
        <li>Вода</li>
        <li>Сок</li>
      </ol>
    </li>
  </ul>
</body>
</html>


6. Ссылки создаются с помощью <a>. Атрибут href задаёт адрес.
Чтобы открыть ссылку в новой вкладке — используйте target="_blank".
Можно ссылаться и на локальные файлы (например, about.html).
Задания:
   1. Создайте ссылку на https://google.com с текстом Google.
   7. Сделайте ссылку, которая откроется в новой вкладке.
   8. Добавьте ссылку на другую страницу about.html.
<!DOCTYPE html>
<html>
<body>
  <a href="https://google.com">Google</a><br>
  <a href="https://google.com" target="_blank">Google в новой вкладке</a><br>
  <a href="about.html">О сайте</a>
</body>
</html>


7.Изображения добавляются с помощью <img>.
   * src — путь к файлу
   * alt — описание (важно для доступности)
   * width / height — ширина/высота
Изображение можно сделать ссылкой, обернув его в <a>.
Задания:
   1. Добавьте картинку cat.jpg с alt="Милый кот".
   9. Установите ширину 200 пикселей.
   10. Сделайте картинку кликабельной, ведущей на Google.
<!DOCTYPE html>
<html>
<body>
  <img src="cat.jpg" alt="Милый кот" width="200">
  <a href="https://google.com">
    <img src="cat.jpg" alt="Кликабельный кот" width="200">
  </a>
</body>
</html>
8. Таблицы:
Таблицы представляют данные в строках и столбцах.
   * <table> — таблица
   * <tr> — строка
   * <th> — заголовок
   * <td> — ячейка данных
Задания:
   1. Создайте таблицу с 2 строками и 2 столбцами.
   11. Добавьте заголовки с помощью <th>.
   12. Добавьте строку с данными с помощью <td>.
<!DOCTYPE html>
<html>
<body>
  <table border="1">
    <tr>
      <th>Имя</th>
      <th>Возраст</th>
    </tr>
    <tr>
      <td>Арсен</td>
      <td>25</td>
    </tr>
  </table>
</body>
</html>




9. Формы позволяют собирать данные от пользователей.
   * <input type="text"> — текстовое поле
   * <input type="password"> — поле пароля
   * <input type="submit"> — кнопка отправки
Задания:
   1. Создайте форму с полем для имени.
   13. Добавьте поле для пароля.
   14. Добавьте кнопку отправки.
<!DOCTYPE html>
<html>
<body>
  <form>
    Имя: <input type="text" name="username"><br>
    Пароль: <input type="password" name="password"><br>
    <input type="submit" value="Отправить">
  </form>
</body>
</html>


10. Семантические теги
Семантические теги описывают смысл содержимого:
   * <header> = верхняя часть (шапка сайта)
   * <nav> = меню навигации
   * <footer> = нижняя часть (подвал сайта)
Это улучшает SEO и доступность.
Задания:
   1. Создайте <header> с названием сайта.
   15. Создайте <nav> с навигационными ссылками.
   16. Добавьте <footer> с текстом об авторских правах.


<!DOCTYPE html>
<html>
<body>
  <header>
    <h1>Мой сайт</h1>
  </header>


  <nav>
    <a href="index.html">Главная</a> |
    <a href="about.html">Обо мне</a>
  </nav>


  <footer>
    <p>&copy; 2025 Мой сайт</p>
  </footer>
</body>
</html>




11. Вы изучили основы HTML: структуру, заголовки, текст, списки, ссылки, изображения, таблицы, формы и семантику.
Теперь создайте полноценную страницу "Обо мне", используя все полученные знания.
Задания:
   1. Создайте HTML-структуру с <!DOCTYPE html>, <html>, <head>, <body>.
   17. Добавьте шапку (<header>) с вашим именем (<h1>) и меню (<nav>).
   18. Напишите абзац о себе, выделите важные слова жирным и курсивом.
   19. Вставьте картинку (например, me.jpg).
   20. Сделайте список хобби.
   21. Создайте таблицу с вашим именем и возрастом.
   22. Добавьте форму для обратной связи (имя, email, сообщение, кнопка).
   23. Внизу сделайте подвал (<footer>).


<!DOCTYPE html>
<html>
<head>
  <title>Обо мне</title>
</head>
<body>


  <!-- Шапка -->
  <header>
    <h1>Иван Иванов</h1>
    <nav>
      <a href="index.html">Главная</a> |
      <a href="#hobbies">Хобби</a> |
      <a href="#contact">Контакты</a>
    </nav>
  </header>


  <!-- Введение -->
  <section>
    <h2>Обо мне</h2>
    <p>Привет! Меня зовут <b>Иван Иванов</b>. Я начинающий веб-разработчик и 
    <i>люблю изучать новые технологии</i>.</p>
  </section>


  <!-- Картинка -->
  <section>
    <h2>Моё фото</h2>
    <img src="me.jpg" alt="Фото обо мне" width="200">
  </section>


  <!-- Хобби -->
  <section id="hobbies">
    <h2>Мои хобби</h2>
    <ul>
      <li>Программирование</li>
      <li>Игра на гитаре</li>
      <li>Путешествия</li>
    </ul>
  </section>


  <!-- Таблица -->
  <section>
    <h2>Основная информация</h2>
    <table border="1">
      <tr>
        <th>Имя</th>
        <td>Иван Иванов</td>
      </tr>
      <tr>
        <th>Возраст</th>
        <td>25</td>
      </tr>
    </table>
  </section>


  <!-- Форма -->
  <section id="contact">
    <h2>Связаться со мной</h2>
    <form>
      Имя: <input type="text" name="name"><br>
      Email: <input type="email" name="email"><br>
      Сообщение: <br><textarea name="message" rows="4" cols="30"></textarea><br>
      <input type="submit" value="Отправить">
    </form>
  </section>


  <!-- Подвал -->
  <footer>
    <p>&copy; 2025 Мой сайт</p>
  </footer>


</body>
</html>
